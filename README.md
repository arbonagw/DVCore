DeepVoid.DVCore
=============

The core gameplay classes for DeepVoid

Overview
--------

DVCore features most of DeepVoid's gameplay, networking, weapons system. You will find the following features :

* Multiplayer deathmatch
* Weapon customization
* Full menu interface and HUD
* Server browsing, player stats through the C "MasterServerBridge" DLL
* And more !

DVCore is *not* a ready-to-use FPS library. It's a part of a complex FPS game, provided for reference.
